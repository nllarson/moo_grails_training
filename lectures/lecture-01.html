Grails...what???

Lecture 01

Grails is an Open Source, full stack, web application framework for the JVM. It takes advantage of the Groovy programming language and convention over configuration to provide a productive and stream-lined development experience.

You can visit the Grails and Groovy websites to learn more about them.

grails.org

groovy.codehaus.org

Grails framework comes with command line support. Your ide (Eclipse, IntelliJ) will use these command line scripts to generate files, configure the application, run the application, etc...

  • Runs on the JVM
  • Built on top of may other frameworks such as Spring and Hibernate
  • Convention of Configuration
